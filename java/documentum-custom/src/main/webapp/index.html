Documentum custom API

File merge

File merge status

Example: http://localhost:8080/documentum-custom/rest/file/merge/status?filePath=%2FTemp%2Fsubmition_0001%2Fexample_file.txt

File merge stop

Example: http://localhost:8080/documentum-custom/rest/file/merge/stop?filePath=%2FTemp%2Fsubmition_0001%2Fexample_file.txt

clear all status

Example: http://localhost:8080/documentum-custom/rest/status/clear
